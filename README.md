## Print Statement
To print anything, use ```print: value```.
> ```print: "Hello World!";``` => **OUTPUT**: Hello World

> ```print: 2127";``` => **OUTPUT**: 2127

> ```print: x;``` => **OUTPUT:** value of variable x

## Variables
To create a variable, use ```variable name = value```
> ```x = 1;```

> ```str = "hello";```
